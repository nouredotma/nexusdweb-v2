## 🏁 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm (standard) or pnpm (recommended)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nouredotma/nexusdweb-v2.git
   cd nexusdweb-v2
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. **Launch Application:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

```
.
├── app/                # Next.js App Router (Pages, Layouts, Metadata)
├── components/         # Atomic & Layout components (UI, Homepage sections)

├── lib/                # Shared utilities, constants, and helper functions
├── public/             # Static assets (Images, Global Icons, OG Images)
└── styles/             # Global CSS and Tailwind directives
```
